# Audio Frequency Comparison and Similarity Evaluation

## Project Description
This project compares two audio recordings and evaluates their similarity using frequency-domain analysis.

## Objective
- Analyze frequency features of two audio files  
- Extract MFCC features  
- Compute similarity score  
- Display comparison graph  

## Tools and Technologies
- Python  
- Librosa  
- NumPy  
- Matplotlib  
- SciPy  
- VS Code  

## Files Included
- audio1.wav  
- audio2.wav  
- audio_similarity.py  

## Output
- Numerical similarity score  
- Frequency comparison graph  

## How to Run
1. Install required libraries  
   pip install librosa numpy matplotlib scipy  

2. Run the program  
   python audio_similarity.py  

## Result
The program calculates similarity between two audio signals and displays graphical comparison.
